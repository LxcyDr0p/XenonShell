# XenonShell

## XenonShell CLI - About
XenonShell is a CLI based Xbox 360 program specifically designed for consoles running modified software. XenonShell is the successor to an earlier and much more baseline program, XeCmd (https://github.com/LxcyDr0p/XeCMD).
